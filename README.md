# 📝 Blog Platform

A full-stack **Blog Platform** where users own their content and can create, edit, and delete only their own posts. The application implements secure authentication and authorization, image uploads with **Cloudinary** and **Multer**, and relational data handling using **Mongoose Populate**.

## 🚀 Features

### 👤 User Features

* User registration and login
* Create blog posts with cover images
* View all published posts
* Read individual blog posts
* Edit and delete only your own posts
* View author information on posts
* Responsive and user-friendly interface

### 🔒 Authorization

* JWT-based authentication
* Protected routes
* Ownership-based authorization
* Only post authors can update or delete their posts

## 🖼️ Image Uploads

* **Multer** for handling multipart/form-data and file uploads
* **Cloudinary** for cloud-based image storage and delivery
* Automatic image management and optimization

## 🔗 Relational Data

* **Mongoose Populate** to fetch related documents
* Display author details alongside blog posts
* Efficient handling of references between collections
